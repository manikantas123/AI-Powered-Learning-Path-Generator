10. AI Engineer Roadmap
Introduction: What is an AI Engineer?, Roles and Responsibilities, Impact on Product Development, AI Engineer vs ML Engineer.

Working with LLMs:

Core LLM Elements: Tokens, Context, Sampling Parameters (Temperature, Top-K, Top-P, Repetition Penalty).

Common Terminology: AI vs ML, Large Language Model (LLM), Embeddings, Training, Inference, Vector DBs, AI Agents, RAG, Fine-tuning, Prompt Engineering, Context Engineering.

Prompt vs Context Engineering:

Prompt Engineering: Zero-shot, Few-shot, FLAN, CoT (Chain of Thought), Input Prompt, Function Calling, Prompt Caching, Streaming Responses, System Prompting.

Context Engineering: External Memory, Fixed and Dynamic Filters, Context Composition, Context Limitation.

Types of Models: Pre-trained Models, Closed vs Open Source, Self-Hosted Models.

Closed: Anthropic Claude, Google Gemini, OpenAI (GPT), Cohere, Mistral.

Open: Meta Llama, DeepSeek, Qwen, Mistral.

Embeddings & Vector Databases:

What are Embeddings: Semantic Search, Data Classification, Recommendation Systems, Anomaly Detection.

Embedding Models: OpenAI Embeddings API, Gemini Embedding, Cohere, Sentence Transformers (Open Source).

Vector Databases: Chroma, Pinecone, Weaviate, FAISS, Milvus, Qdrant, LanceDB, MongoDB Atlas.

RAG (Retrieval-Augmented Generation): Why use RAG?, RAG vs Fine-tuning, Implementing RAG (Chunking, Embedding, Vector Database, Retrieval Process, Generation).

Frameworks: LangChain, LlamaIndex, Haystack, Realflow.

AI Agents: Agent Objectives, ReAct Prompting, Multi-agents, OpenAI Assistants API, Claude Agent SDK.

Model Context Protocol (MCP): MCP Host, MCP Client, MCP Server, Data Layer, Transport Layer.

AI Safety and Ethics: Prompt Injection Attacks, Bias and Fairness, Content Moderation APIs, Adversarial Testing.

Multimodal AI: Image Understanding/Generation, Video Understanding, Audio Processing, Text-to-Speech.

Development Tools: Claude Code, Cursor, Windsurf, Replit.
