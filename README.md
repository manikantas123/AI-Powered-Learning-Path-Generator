# 🚀 AI-Powered Resume Analyzer & Career Roadmap Generator

## 📌 Overview

This is a full-stack web application that analyzes a user’s resume using AI and generates a personalized career roadmap based on a selected job role. It helps students and job seekers understand their current skill level and what they need to improve to achieve their target role.

---

## 🎯 Features

* 📄 Upload resume (PDF format)
* 🎯 Select target job role
* 🤖 AI-based resume analysis
* 📊 Match percentage calculation
* ⚠️ Missing skills identification
* 🗺️ Personalized learning roadmap generation
* 🔐 User authentication using JWT
* 🗂️ Roadmap history tracking

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Add API Key

Update your API key in `app.py`:

```python
API_KEY = "YOUR_GEMINI_API_KEY"
```

### 5️⃣ Run the application

```bash
python app.py
```

### 6️⃣ Open in browser

```
http://127.0.0.1:5000
```
